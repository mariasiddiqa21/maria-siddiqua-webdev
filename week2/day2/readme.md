# 🚀 Data Analysis Dashboard (Drivers & Projects)

## 📌 Overview

This project is a **mini analytics dashboard** built using **HTML, CSS, and JavaScript**. It processes complex nested data and extracts meaningful insights for:

* 🚗 Ride-sharing drivers
* 🏢 Company projects & employees

It simulates how real-world backend systems analyze and present data.

---

## 🎯 Features

### 🚗 Driver Analysis

* Calculates total earnings (only completed rides)
* Computes average rating per driver
* Identifies highest earning driver
* Filters drivers with:

  * At least 5 completed rides
  * Average rating above 4.5
* Detects drivers with all rides incomplete

---

### 🏢 Project Analysis

* Calculates total completed hours per employee
* Computes project cost (based on $10/hour)
* Finds most hardworking employee
* Detects projects with incomplete employees
* Extracts employees working on multiple projects (no duplicates)
